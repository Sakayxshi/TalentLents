# TalentLens

<h2>  Project Idea  </h2>

~Create a fully working app that will take a CSV file with employee database

~Take a prompt from a user -> Where the user tell about a specific Project that the comapny is working on

~App will create a "Team Roster" Dashboard where user will assemble a team needed for the Project based on different Metrics and get 2 scenarios for team based on pre-entered information
